📚 Quick Tutorial

Everything you need to know to get productive fast. All changes auto-save to your chosen file.

Editing Tasks
🖱️
Drag to move & resize
Drag any bar left/right to move it. Drag the right edge to extend or shorten the duration.
✏️
Click to edit
Click any bar or task label to open the edit modal — rename, change status, set ticket IDs, and link dependencies.
⠿
Reorder tasks
Drag the ⠿ handle in the label column to drag tasks up or down within a group.
Adding Content
＋
Quick-add menu
Click the ＋ button in the toolbar to add a new Task, Group, Milestone, or start a fresh Gantt.
📋
Groups & tasks
Tasks belong to one or more groups (swimlanes). Use the + Add Task button inside each group row for quick inline adds.
◆
Milestones
Milestones appear as vertical colored lines across the chart — great for marking key dates or launch targets.
Dependencies
🔗
Blocks / Blocked By
In the task edit modal, link tasks that block each other. Dependencies are always kept in sync bidirectionally.
👁️
Dependency view
Click 🔗 Deps in the toolbar to draw animated lines between dependent tasks. Red = active blocker, green = blocker is done. Use ◉ Only to focus only on tasks with deps.
Views & Filters
↕️
Sort & filter
Use the sort dropdown to order tasks by name, length, or status. Toggle ✓ Hide Done or ⚠ At-Risk Only to focus your view.
📌
Pin label column
Click the 📌 pin icon in the header to freeze the task label column while scrolling horizontally.
Settings & Saving
⚙️
Settings
Click the ⚙ gear icon to edit month columns (add, rename, reorder) or manage custom task statuses with custom colors.
💾
Auto-save
Every change is auto-saved to your browser and to a local file (if you picked one). Use 💾 to choose or update the file location, and 📂 to load a previously saved file.
